# Agora Live

## Overview
**AgoraLive** is an entertainment open source project containing multi scenario. The project includes the following scenario solutions:

* LiveStreaming
    * [Android](Android/scenes/show)
    * [iOS](iOS/AgoraEntScenarios/Scenes/Show)
